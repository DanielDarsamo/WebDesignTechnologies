The application is built with a vanilla tech stack, focusing on core web technologies:

Frontend:

HTML5 for structure
CSS3 with modern features (CSS Variables, Flexbox, Grid, Animations)
Vanilla JavaScript (ES6+) with modular architecture
Key Features:

Mobile-first responsive design
CSS custom properties for theming
Local storage for data persistence
Custom event system
Module pattern for code organization
File Structure:


├── index.html          # Main HTML structure
├── css/
│   └── styles.css      # All styles and animations
├── js/
│   ├── app.js         # Main application logic
│   ├── cart.js        # Cart functionality
│   ├── data.js        # Menu and product data
│   ├── favorites.js   # Favorites system
│   ├── order.js       # Order management
│   ├── translations.js # Language support
│   └── ui.js          # UI controllers and rendering
└── assets/
    └── logo.png       # App logo
